WHAT IS IT?

Hi, welcome on "Run student run" web site.
"Run student run" is a mobile 2D platform game created for everyone wants playing with a game thought like a computer science bachelor degree.
Yes, you read that correctly. A bachelor degree!

In "Run student run" you will be able to "study", in a very different modality. Infact, you will be able to pass all the bachelor degree's exam only taking books scattered in the levels (much more fun than real college study, don't you think?).

And now start with presentations!
